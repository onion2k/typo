# Typo

Typo is a tool for exploring CSS text settings.

# Install and Build

Clone the repo and then run `npm i && npm start` from your terminal.

# Screenshot

![Typo screenshot](/typo.png?raw=true "Typo v0.1")

# Using Typo

## Split Text View

Drag the center bar left and right to view the original or copy text.

## CSS Settings

Click a setting's value to edit. Click the split arrow to the right of the setting to copy it to the other side.
